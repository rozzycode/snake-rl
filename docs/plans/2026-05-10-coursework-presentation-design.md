# 2026-05-10 Coursework Presentation Design

## Goal
Create a defense-ready presentation deck for the coursework on reinforcement learning for Snake, using the finished coursework text, real project screenshots, and measured experiment results.

## Approved design
1. Build an editable `.pptx` deck with 10--12 slides.
2. Use a story-led defense structure rather than a literal chapter-by-chapter retelling.
3. Keep the visual system academic and clean, with clear hierarchy and restrained color use.
4. Use real project materials only:
   - architecture diagram
   - screenshots of manual mode, agent mode, game-over modal
   - training chart
   - real metrics from the coursework text.
5. Cover the whole defense arc:
   - topic and relevance
   - problem statement
   - why Snake and why RL
   - why Q-learning was chosen
   - how the environment is designed
   - how the software is structured
   - what the experiments showed
   - limitations and next steps.

## Target slide structure
1. Title slide
2. Relevance, goal, and tasks
3. Why Snake is a good RL task
4. Why Q-learning was chosen
5. Environment design: field, state, actions, rewards
6. State representation and agent logic
7. Program architecture
8. Interface and modes of operation
9. Experimental results
10. Hyperparameters, limitations, and DQN direction
11. Final conclusions

## Visual system
- 16:9 slides
- light editorial background
- serif-like claim titles plus clean sans body text
- accent color reserved for emphasis
- minimal decorative elements
- screenshots and figures treated as primary proof objects, not filler.
